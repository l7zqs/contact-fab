# 💬 ContactPro - Floating Contact Button (CDN Ready)

A modern, lightweight floating contact button with smooth animations for WhatsApp and Telegram.

✨ Built with pure JavaScript (no dependencies)  
🚀 Easy to use via CDN  
📱 Fully responsive (mobile + desktop)  

---

## 🔥 Features

- Floating action button (FAB)
- Smooth expand animation
- WhatsApp & Telegram quick access
- Pulse effect for modern UI
- No HTML required (auto inject)
- Lightweight & fast
- CDN ready (via jsDelivr)

---

## 📦 Installation (CDN)

Include this script in your website:

```html
<script src="https://cdn.jsdelivr.net/gh/YOURNAME/contact-pro@v1.0.0/contact-pro.js"></script>
```

Then initialize:

```html
<script>
  new ContactPro({
    whatsapp: "8801234567890",
    telegram: "yourusername"
  });
</script>
```

---

## ⚙️ Configuration

| Option     | Description              | Example            |
|------------|--------------------------|--------------------|
| whatsapp   | Your WhatsApp number     | "8801234567890"    |
| telegram   | Your Telegram username   | "yourusername"     |
